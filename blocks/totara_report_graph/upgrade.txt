This files describes API changes in /blocks/totara_report_graph,
information provided here is intended especially for developers.

=== 13.0 ===
* Deprecated ajax_graph.php which was used to generate PDFs for unsupported IE6-IE9 browsers. This file should no longer be used
* Deprecated the following functions in block_totara_report_graph\util functions in favour of util::get_graph.
    * util::get_svg
    * util::get_cache_key
    * util::get_svg_data
    * util::send_svg
    * util::send_pdf

=== 11.0 ===
* The CSS class names below have been replaced with the following:
    - "rb-block-svggraph" => "block_totara_report_graph_svggraph"
